## Github

> Github repositoryfor Langtrace Typescript SDK's -> [Scale3-Labs/langtrace-python-sdk](https://github.com/Scale3-Labs/langtrace-typescript-sdk).

<Warning>
	The name of Typescript SDK has word `langtrase` with a `s` and NOT
	`langtrace`.
</Warning>

## Installation

You can install the Langtrace Typescript SDK using npm:

```bash
npm i @langtrase/typescript-sdk
```

## Initialization

To initialize the Langtrace Typescript SDK

```typescript
import * as Langtrace from "@langtrase/typescript-sdk";
LangTrace.init();
```

For more options on how to use SDK, please refer to [features section](/features).
