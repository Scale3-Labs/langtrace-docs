## Getting Started

<Steps>
  <Step title="Sign Up">
    Create your account at [Langtrace](https://langtrace.ai) and create a new Project to generate your API key.
  </Step>

  <Step title="Installation">
    Install the Langtrace Python SDK using pip:
    ```bash Shell
    pip install langtrace-python-sdk
    ```
  </Step>

  <Step title="Initialize SDK">
    Initialize the SDK with your API key:
    ```python
    # Import it into your project
    from langtrace_python_sdk import langtrace # Must precede any llm module imports
    langtrace.init(api_key = '<LANGTRACE_API_KEY>')
    ```
  </Step>
</Steps>

## Self-Hosted Setup

<Note>
  For self-hosted Langtrace installations, you'll need to specify your custom API host.
</Note>

<CodeGroup>
  ```python Python
  from langtrace_python_sdk import langtrace
  langtrace.init(
    api_key="<YOUR API KEY>",
    api_host="http://localhost:3000/api/trace",
  )
  ```
</CodeGroup>

<Card title="Configuration Note">
  Replace `http://localhost:3000/api/trace` with your own DNS or IP address for self-hosted installations.
</Card>
