## Github

> Github repositoryfor Langtrace Python SDK's -> [Scale3-Labs/langtrace-python-sdk](https://github.com/Scale3-Labs/langtrace-python-sdk).

## Installation

You can install the Langtrace Python SDK using pip:

```bash
pip install langtrace-python-sdk
```

## Initialization

To initialize the Langtrace Python SDK

```python
import langtrace
langtrace.init()
```

For more options on how to use SDK, please refer to [features section](/features).
