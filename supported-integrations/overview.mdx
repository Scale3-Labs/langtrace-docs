| [Perplexity](https://perplexity.ai/)                                                  | LLM             | &#x2714;       | &#x2714;   |
| [Langchain](https://www.langchain.com/)                                               | Framework       | &#x2718;       | &#x2714;   |
| [LlamaIndex](https://www.llamaindex.ai/)                                              | Framework       | &#x2714;       | &#x2714;   |
| [Pinecone](https://www.pinecone.io/)                                                  | Vector Database | &#x2714;       | &#x2714;   |
