Chroma is a AI-native open-source vector database focused on developer productivity and happiness. Chroma is licensed under Apache 2.0.

Install Chroma with:

<CodeGroup>
```bash Python
pip install chromadb
```

```bash Typescript
yarn install chromadb chromadb-default-embed
```

</CodeGroup>
